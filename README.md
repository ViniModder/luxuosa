## ⚙️ Framework

A Base Luxuosa utiliza uma versão personalizada do **vRPex**, otimizada para oferecer:

- 🔹 Desempenho superior
- 🔹 Código assíncrono
- 🔹 Arquitetura modular
- 🔹 Integração nativa com MariaDB
- 🔹 Estabilidade para servidores médios e grandes
